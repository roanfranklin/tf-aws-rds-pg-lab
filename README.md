# tf-aws-rds-pg-lab
Laboratório - Terraform AWS - RDS - PostgreSQL

Execute os comando abaixo para provisionar:

```bash
terraform init
terraform plan
terraform apply
```

## Informações

### Region

 Norte da Virginha ( us-east-1 )


### RDS

 Engine: PostgreSQL

 Versão: 11.5

 Tipo de instância: db.m5.large
